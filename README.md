# Yandex-DataLens-Dashboards
Links to my dashboards built in Yandex BI tool DataLens

## 1. Ted Talks interactive dashboard (link / ссылка : https://datalens.yandex/5jcsz12kmuaur)

An interactive dashboard that helps you look at TED Talks from 1984 to 2021 from different angles. Which topics are the most popular? Which field are the most funny or charismatic speakers from? Here we can find out :)

- Type: analytical
- Tech Stack: Yandex DataLens, calculated fields, filters
- Artefacts: Yandex DataLens dashboard with main page (dashboard itself), and information page (contains additional info on charts and filters)
<br>

Интерактивный дашборд, с помощью которого можно рассмотреть под разными углами конференции Ted Talks 1984 - 2021 годов. Какие темы самые популярные? Спикеры из каких областей самые смешные или харизматичные? Здесь можно найти ответы на эти вопросы :)

- Тип: аналитический
- Стек: Yandex DataLens, вычисляемые поля, фильтры
- Артефакты: дашборд с основной страницей (фильтры, визуализации) и информационной страницей (подробное описание визуализаций и фильтров)

## 2. Food Delivery App static dashboard (link / ссылка: https://datalens.yandex/ddr5mopczr5oy)

A static dashboard to analyze average monthly/daily AOV, DAU and daily order CR for a food delivery app during the period from 1 May 2021 to 30 June 2021

- Type: analytical
- Tech Stack: Yandex DataLens, calculated fields, tooltips
- Artefacts: Yandex DataLens dashboard with 1) main page (key metrics from the dashboard specification, short conclusions), 2) 'AOV decomposition' page (deeper AOV analysis, short conclusions), and 3) conslusions and recommendations page
<br>

Статический дашборд для анализа среднемесячного / дневного AOV, DAU, конверсии в заказ по дням по данным сервиса доставки еды а период с 1 мая 2021 по 30 июня 2021 года

- Тип: аналитический
- Стек: Yandex DataLens, вычисляемые поля, тултипы
- Артефакты: дашборд с 1) основной страницей (ключевые метрики по ТЗ, краткие выводы), 2) страницей с декомпозицией среднего чека (более подробный анализ, краткие выводы) и 3) страницей с подробными выводами и рекомендациями
