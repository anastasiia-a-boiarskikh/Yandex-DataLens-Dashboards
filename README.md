# Yandex-DataLens-Dashboards
Links to my dashboards built in Yandex BI tool DataLens

## 1. Ted Talks interactive dashboard (link / ссылка : https://datalens.yandex/5jcsz12kmuaur)

An interactive dashboard that helps one look at Ted Talks from 1984 to 2021 from different angles. Which topics are the most popular? Speakers from which field are the most funny or charismatic? Here we can find out :)

Интерактивный дашборд, с помощью которого можно рассмотреть под разными углами конференции Ted Talks 1984 - 2021 годов. Какие темы самые популярные? Спикеры из каких областей самые смешные или харизматичные? Здесь можно найти ответы на эти вопросы :)

## 2. Food Delivery App static dashboard (link / ссылка: https://datalens.yandex/ddr5mopczr5oy)

A static dashboard to analyze average monthly/daily AOV, DAU and daily order CR for a food delivery app during the period from 1 May 2021 to 30 June 2021

Статический дашборд для анализа среднемесячного / дневного AOV, DAU, конверсии в заказ по дням по данным сервиса доставки еды а период с 1 мая 2021 по 30 июня 2021 года
